## 📱 كيفية إضافة البروكسي في تيليجرام — خطوة بخطوة

### الطريقة 1 — عبر رابط مباشر (الأسهل)

1. افتح ملف `verified_proxies.txt` عبر الرابط أعلاه
2. انسخ أي سطر — وهو رابط جاهز بصيغة `https://t.me/proxy?server=1.2.3.4&port=443&secret=ee...`
3. الصق الرابط في شريط عنوان المتصفح أو افتحه على هاتفك
4. سيفتح تيليجرام تلقائياً بنافذة **"ربط بروكسي"**
5. اضغط **"اتصال"**

### الطريقة 2 — يدوياً عبر إعدادات تيليجرام

1. افتح ملف البروكسيات، وابحث عن السطر الذي يحتوي على معاملات `server`، `port`، `secret`
2. في تيليجرام: **الإعدادات ← متقدم ← إعدادات الاتصال ← البروكسي**
3. **"إضافة بروكسي"** ← **MTProto**
4. أدخل يدوياً: الخادم، المنفذ، المفتاح السري (من السطر المنسوخ)
5. احفظ وفعّل البروكسي عبر المفتاح

### أندرويد / iOS

يعمل بنفس الطريقة أعلاه: **الإعدادات ← البيانات والتخزين ← إعدادات البروكسي** (أو **متقدم ← البروكسي**، حسب الإصدار) ← **إضافة بروكسي** ← MTProto.

### سطح المكتب (Windows / macOS / Linux)

**الإعدادات ← الإعدادات المتقدمة ← إعدادات الاتصال ← إضافة بروكسي** ← MTProto ← أدخل البيانات يدوياً.
