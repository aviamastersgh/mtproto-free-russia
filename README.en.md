## 📱 How to add a proxy in Telegram — step by step

### Method 1 — direct link (easiest)

1. Open `verified_proxies.txt` via the link above
2. Copy any line — it's a ready-made link like `https://t.me/proxy?server=1.2.3.4&port=443&secret=ee...`
3. Paste the link into your browser's address bar or open it on your phone
4. Telegram will open automatically with a **"Connect Proxy"** prompt
5. Tap **"Connect"**

### Method 2 — manually via Telegram settings

1. Open the proxy file and find a line with `server`, `port`, `secret`
2. In Telegram: **Settings → Advanced → Connection → Proxy**
3. **"Add Proxy"** → **MTProto**
4. Enter manually: server, port, secret (from the copied line)
5. Save and toggle the proxy on

### Android / iOS

Same as above: **Settings → Data and Storage → Proxy Settings** (or **Advanced → Proxy**, depending on the version) → **Add Proxy** → MTProto.

### Desktop (Windows / macOS / Linux)

**Settings → Advanced Settings → Connection Type → Add Proxy** → MTProto → enter the details manually.

## ❓ FAQ

<details>
<summary><strong>I added a proxy but Telegram won't connect. What now?</strong></summary>

1. Make sure you're using `verified_proxies.txt`, not `all_proxies.txt`
2. Try several different lines from the file — not all servers are equally loaded
3. Check that you copied the whole line, including the `secret`
4. Refresh the file manually — proxies update every 2 hours, older ones may drop off

</details>

<details>
<summary><strong>Why did the proxy work yesterday but not today?</strong></summary>

Public free servers have no stability guarantees. They can be blocked by RKN, overloaded by many users, or shut down by the owner.

That's why the list refreshes every 2 hours. Check `verified_proxies.txt` again if your current server stops responding.

</details>

<details>
<summary><strong>Is it safe to use public MTProto proxies?</strong></summary>

Public proxies are free servers run by unknown operators. MTProto encrypts the data, but the server owner could theoretically see connection metadata (e.g. which Telegram server IP you're connecting to).

**Recommendation:** for extra privacy, use a personal key on a dedicated server.

</details>

<details>
<summary><strong>Is an MTProto proxy the same as a VPN?</strong></summary>

No. MTProto proxy only proxies **Telegram traffic** (chats, calls, channels). Other apps — your browser, other messengers — will keep running without any bypass. For a full-device VPN, use [vpn-free-russia](https://github.com/aviamastersgh/vpn-free-russia).

</details>
