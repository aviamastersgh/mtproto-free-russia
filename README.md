## 📱 Как добавить прокси в Telegram — пошаговые инструкции

### Способ 1 — по прямой ссылке (проще всего)

1. Откройте файл `verified_proxies.txt` по ссылке выше
2. Скопируйте любую строку — это готовая ссылка вида `https://t.me/proxy?server=1.2.3.4&port=443&secret=ee...`
3. Вставьте ссылку в адресную строку браузера или откройте её на телефоне
4. Telegram откроется автоматически с окном **«Подключить прокси»**
5. Нажмите **«Подключиться»**

### Способ 2 — вручную через настройки Telegram

1. Откройте файл с прокси, найдите строку с параметрами `server`, `port`, `secret`
2. В Telegram: **Настройки → Дополнительно → Подключение → Прокси**
3. **«Добавить прокси»** → **MTProto**
4. Вставьте вручную: сервер, порт, секрет (из скопированной строки)
5. Сохраните и включите прокси переключателем

### Android / iOS

Работает так же, как выше: **Настройки → Данные и память → Настройки прокси** (или **Дополнительно → Прокси**, в зависимости от версии) → **Добавить прокси** → MTProto.

### Desktop (Windows / macOS / Linux)

**Настройки → Продвинутые настройки → Настройки соединения → Добавить прокси** → MTProto → вставьте данные вручную.

## 🔒 Что такое MTProto

**MTProto Proxy** — собственный протокол Telegram для обхода блокировок, разработанный специально под нужды мессенджера.

| Особенность | Описание |
|-------------|----------|
| **Скорость** | Лёгкий протокол, минимальные задержки |
| **Маскировка** | Трафик похож на обычный TCP-трафик, сложнее для DPI-анализа, чем классический VPN |
| **Область действия** | Работает только внутри Telegram (чаты, звонки, каналы) — не проксирует остальные приложения |
| **Простота** | Подключается одной ссылкой, без установки дополнительных приложений |

## ❓ Частые вопросы

<details>
<summary><strong>Добавил прокси, но Telegram не подключается. Что делать?</strong></summary>

1. Убедитесь, что используете `verified_proxies.txt`, а не `all_proxies.txt`
2. Попробуйте несколько разных строк из файла подряд — не все сервера одинаково загружены
3. Проверьте, что скопировали строку целиком, включая `secret`
4. Обновите файл вручную — прокси обновляются каждые 2 часа, старые могут отключиться

</details>

<details>
<summary><strong>Почему прокси работал вчера, а сегодня нет?</strong></summary>

Публичные бесплатные сервера не имеют гарантий стабильности. Они могут быть заблокированы РКН, перегружены большим числом пользователей или отключены владельцем.

Поэтому список обновляется каждые 2 часа. Проверяйте `verified_proxies.txt` заново, если текущий сервер перестал отвечать.

</details>

<details>
<summary><strong>Безопасно ли использовать публичные MTProto-прокси?</strong></summary>

Публичные прокси — это бесплатные серверы от незнакомых операторов. MTProto шифрует данные, но владелец сервера теоретически может видеть метаданные соединения (например, к какому IP Telegram-сервера вы подключены).

**Рекомендация:** для повышенной приватности используйте личный ключ на выделенном сервере.

</details>

<details>
<summary><strong>MTProto-прокси — это то же самое, что VPN?</strong></summary>

Нет. MTProto-прокси проксирует **только трафик Telegram** (чаты, звонки, каналы). Остальные приложения — браузер, другие мессенджеры — продолжат работать без обхода блокировок. Для полноценного VPN на всё устройство используйте [vpn-free-russia](https://github.com/aviamastersgh/vpn-free-russia).

</details>
