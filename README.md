📖 О проекте

Репозиторий агрегирует публичные MTProto-прокси для Telegram, доступные из России. Скрипт каждые 2 часа:

- Собирает прокси-ссылки из публичных источников
- Убирает дубликаты
- Проверяет доступность серверов (TCP)
- Публикует два файла с актуальными результатами

Регистрация не нужна — просто откройте ссылку в Telegram.

📱 Как подключить прокси в Telegram

1. Откройте файл `verified_proxies.txt`
2. Скопируйте любую ссылку вида `tg://proxy?server=...&port=...&secret=...`
3. Вставьте её в адресную строку браузера или откройте напрямую — Telegram сам предложит подключить прокси
4. Нажмите **Подключить прокси** во всплывающем окне приложения
